# PrestaShop One-Page Checkout Plugin

A **PrestaShop** plugin that transforms the multi-page checkout process into a smooth **one-page experience**.

![App screenshot](https://github.com/aballah-chamakh/oneformcheckout/blob/main/mawlety_one_page_checkout.png)


## Tech Stack
- **Frontend:** HTML,CSS,JS
- **Backend:** vanilla PHP
